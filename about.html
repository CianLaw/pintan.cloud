About

Design-driven
developer & creator

I build things for the web. Sometimes they're useful, sometimes they're just beautiful. Either way, I put my heart into every project.

Every line of code is a decision. Every pixel is a choice. I strive to make both meaningful.

With over 5 years of experience in web development and design, I've had the privilege of working on projects ranging from small indie tools to large-scale applications. My approach is rooted in minimalism — not as an aesthetic choice, but as a philosophy.

I believe that good design is invisible. It doesn't shout for attention; it simply works. The same goes for code — the best code is the code you don't have to think about.

When I'm not coding or designing, you'll find me exploring new technologies, writing about my experiences, or contributing to open-source projects. I'm a lifelong learner, constantly curious about how things work and how they can be made better.
