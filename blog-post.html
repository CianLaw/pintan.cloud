技术 / 2026年7月

从零开始
搭建个人博客

记录我从零搭建个人博客的完整过程，包括技术选型、设计思路和踩过的坑。

一直想拥有一个属于自己的网站，一个完全由自己掌控的空间。今年暑假终于动手了。这篇文章记录了整个过程，从想法到上线。

为什么要做个人博客？

社交媒体虽然方便，但总感觉不是自己的地盘。算法决定别人能不能看到你的内容，平台随时可能改变规则。我想要一个完全由我掌控的地方，想写什么就写什么，想放什么就放什么。

而且，做博客本身就是一个很好的学习过程。前端、后端、部署、域名……每一个环节都能学到东西。

技术选型

我选择了纯静态网站方案。没有后端，没有数据库，所有页面都是 HTML + CSS + JS。好处是部署简单、速度快、安全性高。

三.js 做了首页的粒子特效，给整个网站增加一点动态感。GitHub Pages 做免费托管，阿里云买了域名。

设计思路

整体走温暖简约风。背景用奶白色，强调色用大地色系。导航栏做了毛玻璃效果，滚动时有显示/隐藏动画。首页的粒子会随滚动展开，增加视觉层次。

踩过的坑

最大的坑是 Three.js 的混合模式。一开始用加法混合（AdditiveBlending），在白色背景上粒子全白看不见。后来改成正常混合（NormalBlending）才搞定。

做个人网站最大的收获不只是技术，而是「这是我的地方」的归属感。强烈推荐每个学编程的人都做一个。
